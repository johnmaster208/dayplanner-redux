# DayPlanner-Redux
Version 2 of the Dayplanner app.

Includes
- React-Redux and a few middleware
- Webpack
- Redux Form

## Installation


Clone the repo

```sh
$ git clone https://github.com/johnmaster208/dayplanner-redux.git
```

### Get dependencies

with npm
```sh
$ cd dayplanner-redux
$ npm install
```

or yarn
```sh
$ cd dayplanner-redux
$ yarn install
```

### Run the app

with npm
```sh
$ npm start
```

or yarn
```sh
$ yarn start
```
